1.Метод инициализации данных
------------------------------------------   
Реализация:

Данные инициализируются внутри функции reccurent_rel(n, k):

Создаётся массив F размером n + 1, заполненный нулями.

Начальные условия: F1 = 1, F2 = 1.

ОШИБКИ И ИХ УСТРАНЕНИЕ
-------
def reccurent_rel(n, k):
    F = [0] * (n + 1) 
    F[1] = 1
    F[2] = 1
    
При n < 2 возникает ошибка (например, для n = 0 или n = 1 массив F будет некорректным).

Не обработаны случаи отрицательных значений n или k.

ИСПРАВЛЕНИЕ:
---------
    def reccurent_rel(n, k):
          if n < 1 or k < 0:
            raise ValueError("Некорректные входные данные: n ≥ 1, k ≥ 0")
        
           if n == 1 or n == 2:
                  return 1
        
Добавлены условия для n ≥ 1 и k ≥ 0.

Проверка на отрицательные значения и нули. При неподходящих значениях вызывается исключение ValueError.

### Обработка базовых случаев:

Для n = 1 и n = 2 возвращается 1 без создания массива.

### Оптимизация:

Массив F создаётся только при n ≥ 3.


2.Алгоритм размножения
-------------------------------------------

***Рекуррентное отношение*** – это способ задания членов последовательности с учётом значений предыдущих членов.

Если обозначить через Fn как число пар кроликов на n-ый месяц, то получим ***последовательность Фибоначчи***, в которой члены определяются по формуле Fn =  Fn-1 + Fn-2 
(при этом последовательность начинается с членов F1 = F2 = 1).

F[i-1] - пары кроликов, которые уже были в предыдущем месяце,

k * F[i−2] - новые пары, произведённые зрелыми кроликами (каждая пара производит k новых пар через месяц).

     for i in range(3, n + 1):
           F[i] = F[i - 1] + (k * F[i - 2])
